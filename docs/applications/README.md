# Applications

This section contains documentation for each application in the Sacred Vows monorepo.

## Applications

### [Builder](./builder/README.md)

- [Environment Setup](./builder/env-setup.md) - Configuration for different environments

### [API Server](./api-go/README.md)

- [Environment Setup](./api-go/env-setup.md) - Environment configuration guide

### [Edge Worker](./edge-worker/README.md)
Cloudflare Workers service that serves published invitation sites directly from R2 storage.
