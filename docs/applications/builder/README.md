# Builder Application

The Builder is a React/Vite-based frontend application that provides an intuitive interface for creating and customizing wedding invitations.

## Features

- Real-time preview of invitation changes
- Layout-based architecture for flexible designs
- Multi-language support
- Asset management and upload
- Theme customization
- RSVP configuration

## Quick Start

1. Set up environment variables (see [Environment Setup](./env-setup.md))
2. Install dependencies: `npm install`
3. Start development server: `npm run dev`
4. Open http://localhost:5173
