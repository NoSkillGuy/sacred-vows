# Observability

This section covers monitoring, telemetry, and observability for Sacred Vows.

## Documentation

- [Telemetry](./telemetry.md) - OpenTelemetry implementation and configuration

## Observability Stack

- **Tempo**: Distributed tracing backend
- **Prometheus**: Metrics collection
- **Grafana**: Visualization and dashboards

## Features

- Request ID and trace correlation
- Distributed tracing across services
- RED metrics (Rate, Errors, Duration)
- Vendor-neutral OTLP protocol
