# Prompts Directory

This directory contains AI prompts and personas used for development workflows in this project.

## Purpose

The prompts in this directory define specific personas and workflows for AI-assisted development. These prompts ensure consistent, high-quality development practices across the project.

## Available Prompts

### `software-engineer.md`

A comprehensive prompt defining a Senior Software Engineer persona with:
- 15 years of experience at top-tier companies (Apple, Google)
- Full-stack development expertise
- Strict adherence to Clean Architecture principles
- Test-Driven Development (TDD) methodology
- Detailed planning and documentation practices
- Comprehensive testing approach (unit, integration, E2E)
- Responsive frontend development with cross-device testing
- Thorough documentation maintenance

## Usage

These prompts can be used with AI coding assistants to ensure consistent development practices and quality standards throughout the project lifecycle.

## Structure

Each prompt file contains:
- Professional profile and experience
- Core principles and methodologies
- Detailed workflow processes
- Quality standards and checklists
- Communication guidelines
