## Visual Identity

### Logo

The Sacred Vows logo consists of:
1. **Icon**: A stylized wedding ring with diamond accent
2. **Wordmark**: "Sacred Vows" in Cormorant Garamond

#### Logo Usage Rules
- Maintain minimum clear space equal to the height of the "S"
- Never distort, rotate, or alter the logo proportions
- Use appropriate color variants for different backgrounds
- Minimum size: 32px height for digital, 0.5" for print

#### Logo Variants
- **Primary**: Gold icon with dark text on light backgrounds
- **Reversed**: Light icon and text on dark backgrounds
- **Monochrome**: Single color for special applications

### Color System

See [Color Palette](./color-palette.md) for complete specifications.

**Primary Palette**:

- Rose `#e8b4b8` - Romance, interaction
- Burgundy `#8b2942` - Depth, premium text

See [Typography](./typography.md) for complete specifications.

- **Quicksand** - Body text (sans-serif)
- **Great Vibes** - Decorative accents (script)

## Brand Application

### Digital Products

#### Website
- Cream backgrounds with white cards
- Gold accent for primary actions
- Rose accent for secondary interactions
- Subtle animations (floating petals, gradient shifts)
- Glass-morphism for navigation elements

#### Invitation Layouts
- Each layout maintains brand essence while offering variety
- Color palettes coordinate with wedding themes
- Typography follows hierarchy guidelines
- Premium features use gold accents

### Marketing Materials

#### Social Media
- Consistent color palette across platforms
- Romantic, aspirational imagery
- Couples' stories and testimonials
- Behind-the-scenes team content

#### Email Communications
- Clean, elegant layout
- Personalized greetings
- Clear CTAs with gold buttons
- Consistent footer with branding

### Photography Style

- Natural lighting preferred
- Warm, romantic color grading
- Focus on genuine emotion
- Diverse representation of couples
- Elegant venues and details

## Do's and Don'ts

### Design Do's
✓ Use generous whitespace
✓ Maintain consistent color usage
✓ Apply subtle, purposeful animations
✓ Prioritize readability and accessibility
✓ Use high-quality imagery

### Design Don'ts
✗ Overcrowd layouts with elements
✗ Use colors outside the brand palette
✗ Add distracting or excessive animations
✗ Compromise accessibility for aesthetics
✗ Use low-quality or generic stock photos

### Communication Do's
✓ Lead with empathy
✓ Celebrate the couple's journey
✓ Be helpful and solution-oriented
✓ Personalize when possible
✓ Keep it concise and clear

### Communication Don'ts
✗ Use pushy sales tactics
✗ Make assumptions about couples
✗ Use jargon or technical terms
✗ Be negative or discouraging
✗ Over-promise on features
